# Users
Status: completed
## 1) What this module is
Users is where you manage employee access to the CRM. It stores active staff accounts and basic account status.

## 2) What staff can do here
- View the list of current staff accounts.
- Activate, deactivate, or review user records.
- Confirm each user has the right status for daily work.

## 3) How to use (step-by-step)
1. Open **Users**.
![Screenshot](/docs/screenshots/20-users/01-users-list.png)
2. Search for the staff member by name or email.
3. Open the user record.
4. Check status, team details, and assigned access level.
5. Make the needed update.
6. Select **Save**.
7. Inform the staff member if the change affects their login.

## 4) Important buttons/options
- **New**: Adds a user record.
- **Save**: Confirms account updates.
- **Enable / Disable**: Turns account access on or off.
- **Search / Filter**: Finds users quickly.

## 5) Daily best practices
- Deactivate access quickly when someone leaves or changes role.
- Review user lists regularly to keep records clean.
- Use official work email addresses only.

## 6) Common mistakes to avoid
- Leaving inactive staff with active access.
- Creating duplicate user records.
- Making account changes without notifying relevant managers.
- Updating the wrong user due to similar names.
