# Office Hours
Status: pending

## 1) What this module is
This module is where your team sets working hours. It tells staff and customers when your office is open and when it is closed.

## 2) What staff can do here
- Create office hour schedules for weekdays and weekends
- Set different hours for different teams or locations
- Add special closed days (for example, holidays)
- Update existing schedules when timings change
- Mark one schedule as the main/default schedule

## 3) How to use (step-by-step)
1. Open **Office Hours** from the main menu.
2. Click **New** (or **Add Office Hours**).
3. Enter a clear schedule name, like `Main Office Hours`.
4. Set start and end time for each day.
5. Turn off days when your office is closed.
6. Add special dates (holidays or half-days), if needed.
7. Save the schedule.
8. Set it as **Default** if this should be your main office timing.

## 4) Important buttons/options
- **New / Add Office Hours**: Create a new schedule
- **Save**: Keep your changes
- **Set as Default**: Make this the main schedule used by the system
- **Edit**: Change an existing schedule
- **Disable / Archive**: Stop using an old schedule without deleting history

## 5) Daily best practices
- Check that today’s open/close time is correct.
- Update holiday entries before the holiday week starts.
- Use clear names for schedules (team or location name).
- Keep one active default schedule to avoid confusion.
- Inform staff when office hours are changed.

## 6) Common mistakes to avoid
- Forgetting to save after changing timings
- Setting the wrong AM/PM time
- Creating multiple default schedules
- Not adding holiday closures in advance
- Editing the wrong schedule because names are unclear
