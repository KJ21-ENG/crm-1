# Profile
Status: completed
## 1) What this module is
Profile is where you manage your personal work details and account preferences.

## 2) What staff can do here
- Update contact details
- Change password
- Set notification preferences
- Check role-related account information
- Keep personal details accurate for team communication

## 3) How to use (step-by-step)
1. Open your Profile from the account menu.
Screenshot Path: `apps/crm/docs/documentation creation task screenshots/18-profile/01-profile-settings.png`
Placement: Insert this screenshot directly below the step: "1. Open your Profile from the account menu." (anchor line at insertion time: 15).
2. Review your name, phone number, and email.
3. Update any outdated information.
4. Change your password if needed.
5. Adjust notifications so you receive important alerts.
6. Save all changes.
7. Reopen the profile to confirm updates were saved.

## 4) Important buttons/options
- Edit Profile: Allows changes to personal details.
- Save Changes: Applies your updates.
- Change Password: Updates login password.
- Notification Settings: Controls alert preferences.
- Cancel: Exits without saving.

## 5) Daily best practices
- Keep your phone and email up to date.
- Review notification settings weekly.
- Use a secure password and change it regularly.
- Check your profile after any team or role change.
- Confirm saved changes before leaving the page.

## 6) Common mistakes to avoid
- Forgetting to save after editing.
- Using outdated contact details.
- Turning off all alerts and missing urgent updates.
- Choosing a weak password.
- Assuming changes saved without confirming.
