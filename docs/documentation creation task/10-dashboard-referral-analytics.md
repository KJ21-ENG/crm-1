# 10. Dashboard Referral Analytics
Status: completed
## 1) What this module is
Referral Analytics shows which referral sources bring in the best results and how those referrals move through your conversion journey.

## 2) What staff can do here
- See top-performing referral sources
- Review referral conversion stages (funnel)
- Drill into source-level details for a selected time period
- Export referral analytics for reporting and team reviews

## 3) How to use (step-by-step)
1. Open **Dashboard** and go to **Referral Analytics**.
2. Choose the time range (daily/weekly/monthly/custom).

## 4) Important buttons/options
- **Date range controls**: Set the period you want to review
- **Source table**: Compare referral channels
- **Funnel view**: See conversion stage by stage
- **Details/Drilldown**: Inspect one source deeply
- **Export**: Download referral analytics data

## 5) Daily best practices
- Track top referral sources regularly, not only at month-end
- Watch funnel drop-offs and act quickly on weak stages
- Share weekly referral insights with sales/support teams
- Keep date ranges consistent when comparing sources

## 6) Common mistakes to avoid
- Judging source quality using only lead count (ignore conversion quality)
- Reviewing referrals without checking funnel drop-off points
- Exporting data without first applying the right date filters
- Making channel decisions from one short period only
