| `Ctrl+/` | Show keyboard shortcuts |
| `Esc` | Close modal/dialog |
| `Enter` | Submit form |
| `↑/↓` | Navigate lists |

## Universal Search

Press `Ctrl+K` to search across:
- Leads (by name, mobile)
- Tickets (by subject)
- Customers (by name, Client ID)
- Tasks (by title)

### Search Tips

- Partial matches work
- Search by mobile number
- Search by Client ID

## Bulk Actions

### Select Multiple Items

1. Click checkbox on first item
2. Shift+Click on last item
3. All items in between selected

### Available Bulk Actions

| Action | Description |
|--------|-------------|
| Change Status | Update status on selected |
| Assign | Reassign selected items |
| Export | Download as CSV |
| Delete | Remove selected (admin only) |

## Quick Filters

### Saved Filters

1. Apply filter criteria
2. Click **Save Filter**
3. Name your filter
4. Access from filter dropdown

### Common Filters

| Filter | Use Case |
|--------|----------|
| My Leads | Only my assigned leads |
| Open Today | Items created today |
| Overdue | Past due date |

## Activity Shortcuts

### Quick Note

1. Open any lead/ticket
2. Type note in activity area
3. Press `Enter`
4. Note added instantly

### Quick Call Log

From lead detail:
1. Click phone icon
2. Log outcome after call
3. Activity auto-recorded

## Navigation Tips

### Back Navigation

- Click browser back
- Or use breadcrumbs at top

### Recent Items

System remembers recent:
- Last 10 viewed records
- Quick access from search

### Pinned Items

Pin frequently accessed items:
1. Open the record
2. Click star/pin icon
3. Access from dashboard

## Date Picker Tips

### Common Shortcuts

| Input | Result |
|-------|--------|
| `today` | Today's date |
| `tomorrow` | Tomorrow |
| `+7d` | 7 days from now |
| `next monday` | Coming Monday |

### Custom Range

Click custom range for:
- Start date picker
- End date picker

## Performance Tips

### Reduce Load Times

- Use filters to limit results
- Avoid loading all records
- Use pagination

### Browser Recommendations

- Chrome (recommended)
- Firefox
- Edge

### Clear Cache

If experiencing issues:
1. Clear browser cache
2. Refresh page (`Ctrl+F5`)

## Collaboration Tips

In notes, type `@` to mention:
- Team members get notified
- Creates activity trail

### Comments

Add comments on any record:
- @ mention for attention
- Visible in activity feed

## Security Tips

### Session Management

- Sessions auto-expire
- Re-login if idle
- Logout when leaving desk

### Sensitive Data

- Client data masked by default
- Click to reveal (if permitted)
- Audit trail on access
