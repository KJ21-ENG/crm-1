# 09. Dashboard Analytics
Status: completed
## 1) What this module is
Dashboard Analytics gives a big-picture view of CRM performance. It combines lead, ticket, task, call, activity, and trend information in one place.
*(Note: Access to the main Analytics tab is restricted to Admin users.)*

## 2) What staff can do here
- Monitor overall team performance
- Track daily, weekly, monthly, or custom date-range results
- Compare activity trends over time
- Review recent activities and quick actions
- Spot areas that need immediate follow-up

## 3) How to use (step-by-step)
1. Open **Dashboard** and go to the **Analytics** area.
2. Select a view period: **Daily**, **Weekly**, **Monthly**, or **Custom**.
3. Check the overview numbers first to understand current performance.
![Screenshot](/docs/screenshots/09-dashboard-analytics/01-dashboard-analytics-overview.png)
4. Review each section (Leads, Tickets, Tasks, Calls, Trends).
![Screenshot](/docs/screenshots/09-dashboard-analytics/02-dashboard-analytics-trends.png)
5. Use filters/date controls if numbers look too broad.
6. Open related records from cards/widgets when deeper action is needed.

## 4) Important buttons/options / Dashboard Metrics
- **Daily / Weekly / Monthly / Custom**: Time window controls to filter dashboard data.
- **Date range selector**: Focus on a specific custom period.

**Dashboard Metrics Explained:**
- **Total Leads**: Shows the total number of Leads handled. Includes a breakdown of leads *Created* vs *Assigned* to the user.
- **Total Tickets**: Shows the total number of support Tickets. Includes a breakdown of tickets *Created* vs *Assigned* to the user.
- **Total Tasks**: Shows the total number of Tasks. Includes a breakdown of tasks *Created* vs *Assigned* to the user.
- **Call Logs (Calls Made)**: Shows the total number of calls made and the combined *Total Duration* of those calls.
- **Account Opened**: Shows the total number of leads that were successfully converted into accounts.
- **Account Activated**: Shows the total number of active customer accounts.
- **Recent Activity**: Latest actions and updates.
- **Quick Actions**: Fast shortcuts to common tasks.

## 5) Daily best practices
- Start each day with a quick dashboard scan
- Use the same time range when comparing performance
- Investigate sudden drops or spikes immediately
- Turn dashboard insights into clear action items for the team

## 6) Common mistakes to avoid
- Comparing metrics from different time windows
- Focusing only on totals and ignoring trend direction
- Ignoring recent activity signals that need follow-up
- Taking decisions from one metric without checking related sections
