# Global Search
Status: completed
## 1) What this module is
Global Search helps you find information quickly across the system from one search box.

## 2) What staff can do here
- Find leads, customers, tickets, tasks, and notes
- Jump directly to a specific record
- Search by name, phone number, email, or keyword
- Reuse recent searches to save time

## 3) How to use (step-by-step)
1. Open the search bar at the top of the screen.
2. Type a clear keyword, name, number, or email.
3. Wait for matching results to appear.
4. Review the result type (lead, ticket, task, and so on).
![Screenshot](/docs/screenshots/14-global-search/01-global-search-results.png)
5. Click the correct result to open it.
6. If results are too broad, add one more specific word.
7. If needed, clear the search and try again with a different term.

## 4) Important buttons/options
- Search Box: Where you enter your keyword.
- Recent Searches: Reopen something you searched before.
- Clear: Removes current search text.
- Filters or Categories: Limits results to a specific type.
- Open Result: Takes you straight to the selected record.

## 5) Daily best practices
- Use specific terms (full name, full phone, exact email).
- Check spelling before assuming a record does not exist.
- Use recent searches for repeat lookups.
- Confirm the record type before opening.
- Search first before creating a new record to avoid duplicates.

## 6) Common mistakes to avoid
- Using very short words that return too many results.
- Misspelling names and deciding the record is missing.
- Opening the wrong person because names are similar.
- Creating a duplicate record without searching carefully.
- Forgetting to clear old search text before a new search.
